# **Scripts**

### Running the frontend-code

---

**clone the repo**

```bash
git clone https://github.com/Sam44323/hybrid-workflow.git
```

---

**go to the frontend-folder**

```bash
cd hackernews-search
```

---

**Install the dependencies**

```bash
yarn
```

---

**Running the dev-server**

```bash
yarn dev
```

---

### Running the backend-code

---

**clone the repo**

```bash
git clone https://github.com/Sam44323/hybrid-workflow.git
```

---

**go to the frontend-folder**

```bash
cd ecomm-backend
```

---

**Install the dependencies**

```bash
yarn
```

---

**populate the env's**

```bash
cp .example.env .env

```

---

**Running the dev-server**

```bash
yarn start
```
